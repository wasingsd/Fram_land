# Farm Land Simulator Prototype

A modern, mobile-first frontend prototype for a farm and land management system.

## Features
- **Dashboard**: KPI summary and insights.
- **Map Overview**: Interactive zone management.
- **Add Transaction**: Quick activity logging.
- **Reports**: Financial and operational analysis.
- **Assets**: Resource tracking.

## Tech Stack
- [Next.js](https://nextjs.org) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4.0](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/)

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
